Um pouco sobre cada área que estou aprendendo

HTML

HyperText Markup Language, traduzindo, Linguagem de Marcação de Hipertexto, é utilizada para a estruturação de páginas web, criando documentos que podem ser lidos pela maioria dos navegadores existentes, utilizando tags, onde cada uma tem sua função. Sua versão mais recente é o HTML5.

CSS

Cascading Style Sheets, traduzindo, Folha de Estilo em Cascatas, é utilizada para dar estilo aos códigos/tags do HTML, XML ou XHTML, sendo responsável pela personalização da parte visual do código.
